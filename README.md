🤖 AI-Enabled Campus Help Desk

An AI-Enabled Campus Help Desk is a smart web-based system that automates student support services using Artificial Intelligence. It provides instant responses to student queries, reducing manual workload and improving efficiency in campus communication.

📌 Overview

This project acts as a virtual assistant for students, capable of answering queries related to:

Academics
Admissions
Fees
Timetable
Hostel
Transport
Exams and results

The system is available 24/7, ensuring quick and reliable support without requiring human intervention for common queries.

🚀 Features
🧠 AI Chatbot
Uses NLP to understand and respond to student queries
Provides accurate and contextual answers
⚡ Instant Response System
Real-time replies for better user experience
🔐 User Authentication
Secure login and signup system
📊 Admin Dashboard
Manage FAQs and queries
Monitor system usage
📂 Multi-Category Query Handling
Covers multiple campus-related domains
👨‍💼 Human Support Fallback
Redirects complex queries to admin/staff

🎯 Objectives
Automate campus help desk services
Reduce workload on administrative staff
Provide instant and accurate information
Improve student experience

💡 Use Cases
Students can get answers instantly
Reduces long queues at help desk
Improves communication efficiency
Works anytime (24/7 availability)
